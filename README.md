# X (Twitter) 关注状态检测助手

这是一个简单的 Chrome 扩展程序，用于在 X (Twitter) 页面上自动标注关注状态。

## 功能特点
- **双向关注**：你关注的人也关注了你，背景标记为 **浅绿色**。
- **未回关**：你关注的人没有回关你，背景标记为 **浅红色**。
- **隐私保护**：仅在本地运行，不收集任何数据。

## 安装方法
1. 下载本项目代码到本地。
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式”**。
4. 点击 **“加载已解压的扩展程序”**，选择本项目文件夹。
